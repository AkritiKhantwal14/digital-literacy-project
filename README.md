Project Overview
As a Student Digital Ambassador, I have developed this portfolio to demonstrate my proficiency in navigating the digital landscape. This project covers five key modules of digital literacy: professional branding, coding proficiency, cloud collaboration, digital communication, and cybersecurity. Each task within this repository serves as a resource to help my peers transition into professional digital citizens.

Module Summaries
Task 1: Digital Literacy Presentation
Focus: Visual communication of digital literacy concepts.

Content: A one-page infographic designed in Canva covering professional presence, safe internet practices, and basic definitions.
